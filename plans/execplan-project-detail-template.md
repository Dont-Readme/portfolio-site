# ExecPlan — Project Detail Template

## 목적

프로젝트 상세 8개 섹션 템플릿을 구현하고 유지한다.

## In Scope

- 프로젝트 한 줄 요약
- 배경 / 문제
- 목표
- 내 역할
- 기획 과정
- 주요 화면 또는 이미지
- 핵심 기능
- 결과 / 성과

## Out of Scope

- 배운 점 고정 섹션
- 관련 링크 고정 섹션

## 위험 / 전제

- 모든 프로젝트가 동일한 기본 구조를 따라야 한다.
- 실제 프로젝트 자료가 들어오면 섹션 제목을 유지해야 한다.

## 검증

- MDX 3개 샘플의 섹션 순서 확인
- `/projects/{slug}` 렌더링 확인
- `pnpm build`
