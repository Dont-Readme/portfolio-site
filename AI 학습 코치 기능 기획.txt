<추천 생성 시기>
- 신규 유저: 가입 및 진단 완료 후 최초 생성
- 기존 유저: 해당 날짜의 첫 홈 진입 또는 첫 문제 풀이 완료 중 먼저 발생한 시점
- 당일 세트 존재: 신규 생성 없이 저장된 후보군 사용
- 날짜 변경: 전일 세트 만료 후 신규 생성
<학습 행동 프로필 활용>
- 최신 학습 행동 프로필이 존재하면 시스템이 유형 적합도와 시간 적합도를 계산. AI에는 시스템이 계산한 적합도 점수와 추천 문구 작성에 필요한 근거 사실만 전달
- 행동 프로필이 없으면 유형·시간 적합도는 null로 제외하고 목표·수준 적합도와 취약도를 이용해 추천을 생성
- 추천 생성 시점의 프로필을 해당 날짜 추천 세트에 적용. 당일에 프로필 변경이 있다면 다음 날짜 신규 추천 생성에 프로필 변경 정보 적용"	최초 추천 요청 시 추천 후보군이 저장되고 동일 날짜에 추가 요청 시 새 후보군이 중복 생성되지 않음	
REQ-006	추천	공통	[AI 활용 추천 후보군 선정 및 저장] 콘텐츠별 개인 적합도를 계산하고, 취약도와 결합한 추천 점수로 당일 추천 후보군 최대 10개 생성	추천 가능 콘텐츠 DB와 콘텐츠가 존재	시스템은 후보별 유형 적합도, 시간 적합도, 수준 적합도를 규칙으로 계산. AI는 최대 10개 후보를 한 번의 배치 호출로 평가하여 사용자 목표와 콘텐츠 학습 목표의 의미적 관계, 추천 이유 유형, 학생용 추천 문구를 반환. 시스템은 AI가 판단한 목표 관계를 목표 적합도 점수로 변환하고, 적용 가능한 네 항목을 100점으로 환산하여 개인 적합도를 계산. 취약도는 개인 적합도에 포함하지 않으며, 시스템이 개인 적합도 50%와 취약도 50%를 결합해 최종 점수/순위/다양성을 결정.	후보별 적합도 점수와 계산 주체, 개인 적합도, 최초 추천 점수, 추천 이유가 저장되고 최종 상위 추천 3개가 제공
